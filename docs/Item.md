# ITEM
### Item properties 
```java
class Item{
    private String id;
    private String name;
    private String qty;
    private String price;
}
```
### Item create 
```cmd
item create <id> <name> <qty> <price>
```
### Item find by id
```cmd
item find <id>
```
### Item find all
```cmd
item find all
```
### Item search 
```cmd
item search <key> <value>
```
