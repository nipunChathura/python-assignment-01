# ITEM
### Item properties 
```python
class Item:
    def __init__(self,item_id,name,qty,price):
        self.item_id = item_id
        self.name = name
        self.qty = qty
        self.price = price
```
### Item create 
```cmd
self create <id> <name> <qty> <price>
```
### Item find by id
```cmd
self find <id>
```
### Item find all
```cmd
self find all
```
### Item search 
```cmd
self search <key> <value>
```
